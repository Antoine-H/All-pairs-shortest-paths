Introduction

The component LWindow provides support for managing windows (open, close, resize, move, maximize, minimize...) and installing event handlers (close, activate, mouse down...).

API Reference

Create       LWindow_TWindowType inWindowType Creates a new window of the specified type. The following types are currently available: REGULAR (just that, a regular window with no extra features); FLOATING (a "palette" window, exact features to be determined yet...). InStyle is an OR combination of LWindow_TStyle parameters (LWindow_MinMaxBox has no effect on FLOATING windows). LWindow* CANT_CREATE

Close LWindow* inWindow Sends a CLOSE event to the specified window. Whether the window will close or not depends on the event handler currently installed. If no handler for the CLOSE event is installed, the window will close.
