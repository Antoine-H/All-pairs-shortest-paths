Introduction

The component LStatusBar provides support for creating and managing StatusBar widgets.

SetPart LStatusBar* inBar   Allows the user to specify a new right margin for a status bar part (absolute coordinate in pixels from left edge of parent window's client area). inPartIndex is zero based. To make changes visibile, after one or more SetPart(...) calls, the Refresh(...) function should be called. void OUT_OF_RANGE if inPartIndex is not in the interval [0, numparts-1].
ui1 inPartIndex
ui4 inPartRightMargin

ui1 inPartIndex
i1* inText
