Introduction

The component LMenu provides support for the creation and management of menus.

API Reference

Function Arguments Description Returns Throws
Create       i1* inMenuName Creates a new menu, initially empty. Please note that the character '&' has a special meaning in inMenuName: it will not be displayed, the character immediately following will be underlined, and pressing ALT+"char following &" will open the menu. LMenu* CANT_CREATE
AppendMenuItem        
LMenu* inMenu  

Appends a menu item to a specific menu. inItemHandler can be null (means are provided for installing a handler later). inShortcut (if not null) should be a pointer to a string containing a single capital letter (for example "S"). The itemID will identify the menu item uniquely. Please note that the character '&' has a special meaning in inItemName: it will not be displayed, the character immediately following will be underlined, and pressing ALT+"char following &" will select the menu item (only if the relevant menu is open).
