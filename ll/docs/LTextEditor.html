Introduction

The component LTextEditor provides a fully featured, multi-purpose text editor. An LTextEditor object is not a window, but a widget (multiple editors can therefore be installed in the same window). Standard clipboard operations are supported. A sophisticated event-handling mechanism is provided, greatly increasing flexibility (the editor can easily become a console window or a read-only text viewer...).
