Introduction

This component is a collection of utilities useful in the manipulation of graphs.

f4 inPi
ui4 inSeed
Bool inDirected 		Returns a random LGraph with inN nodes and inC strongly connected clusters. There is an edge in the same cluster with probability inPc, and there is an edge between different clusters with probability inPi. To make sure that clusters are strongly connected, a cycle through all nodes in the cluster is added.

LGraphGen_LoadXML i1* inPathName
LGraph** outGraph
LEdgeInfo** outEdgeInfo		 Loads a file stored in XML format. The file is parsed with the following Document Type Definition:
<!DOCTYPE GRAPH [
<!ELEMENT GRAPH (EDGE*)>
<!ATTLIST GRAPH
    label CDATA #IMPLIED
    nodes CDATA #REQUIRED
    edges CDATA #REQUIRED>
<!ELEMENT EDGE EMPTY>
<!ATTLIST EDGE
    label CDATA #IMPLIED
    weight CDATA #IMPLIED
    source CDATA #REQUIRED
    dest CDATA #REQUIRED>
]>
 The pointers to the loaded structures are in outGraph and in outEdgeInfo.

Bool


Returns TRUE if the parsing was succesful, FALSE otherwise. 		-

LGraphGen_BuildShortestPathTree LGraph* inGraph
LNodeInfo* inPredArray
LNodeInfo** outTreeToGraph
LNodeInfo** outGraphToTree 		Builds a shortest path tree based on source inGraph and passed inPredArray. If different from NULL, after the call of the method, in outTreeToGraph there's the mapping Tree->Graph while in outGraphToTree there's the mapping Graph->Tree.
Note that the LNodeInfo_New for both outTreeToGraph and outGraphToTree is called INSIDE this function. In order to avoid memory leaks, user MUST NOT call the LNodeInfo_New.

inGraph*


Pointer to the created shortest path tree 		-
LGraphGen_CopyGraph LGraph* inGraph Returns a graph that is the copy of inGraph. The copy is only topological. Caller is responsible of the deallocation of returned LGraph*

inGraph*

LGraphGen_ReverseGraph LGraph* inGraph Returns a graph that is the reversed of inGraph. The reversal is only topological. Caller is responsible of the deallocation of returned LGraph*. If inGraph is not directed, the function returns NULL since the operation requested has meaning only for undirected graphs.
