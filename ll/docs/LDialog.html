Introduction

The component LDialog provides pre-made easy-to-use dialog boxes adaptable to most common situations (notifications to the user, questions that require a yes/no answer, opening and saving files...).

GetSaveFileName        
LWindow* inOwner  

Displays a "Save as" dialog box. inFilters is an array that alternates filter descriptions and specifications (e.g. inFilters[0]="C and C++ Source Files" and inFilters[1]="*.C;*.CPP"). inNumFilters specifies the number of description-specification pairs in the array. If the user cancels the operation the function returns False, otherwise it returns True and the complete path plus filename is copied onto inBuffer. The buffer pointed to by inBuffer may also contain a default filename (which might be overwritten by the user's selection). If inOwner is not NULL, the corresponding window will be brought to the foreground before the dialog box is displayed.
