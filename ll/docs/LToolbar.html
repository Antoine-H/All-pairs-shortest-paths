Introduction

The component LToolbar provides support for creating and managing Toolbar widgets.

API Reference

Function Arguments Description Returns Throws
Create       ui2 inNumButtons Creates a toolbar with the specified attributes. Each button has a 16x16 pixel bitmap painted on it (unless it is a separator). InBitmap contains all the button images required by the toolbar, arranged in a rectangular array which is 16 pixels high and a multiple of 16 pixels wide. inButtonArray is a pointer to an array of LToolbar_TButton structs that has inNumButtons elements. A newly created toolbar is always visible (i.e. on top of all other toolbars present). Please note that if the style LToolbar_SEPARATOR is specified for a given button, then the mBitmapIndex and mItemID fields for that button should both be set to zero. LToolbar* CANT_CREATE
