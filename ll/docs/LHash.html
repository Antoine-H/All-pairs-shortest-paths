Introduction

The component LHash provides support for maintaining an hash table that is dynamically expanded or shrinked in order to have a good balance between access time and memory usage. The hash function works on 32-bit keys. Member methods allow to insert, remove, find items. The items used in the table are 32-bit long, so the user can handle them at his needs (i.e. uses different data types).
